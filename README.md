Registration Form

Este projeto é um formulário de registro moderno desenvolvido com **HTML, CSS e JavaScript**, utilizando validação em tempo real, animações e exibição dinâmica dos cadastros.  
O objetivo é demonstrar práticas de design, interatividade e organização de dados no front-end.


 Tecnologias Utilizadas
- HTML — estrutura da página  
- CSS — estilos, animações e responsividade  
- JavaScript — validação, manipulação do DOM e lógica do sistema  
- Bootstrap  — estilização complementar



 Funcionalidades
- Validação em tempo real para:
  - Nome  
  - Idade  
  - Curso  
  - Email  
  - Senha  
- Animações ao digitar e ao enviar  
- Contador de cadastros no topo  
- Exibição dos registros salvos em cards  
- Card de confirmação após o cadastro  
- Ocultação da senha com •  
- Alert de sucesso personalizado  
- Lista de cadastros armazenada em memória  
- Layout estilizado com degradê e sombras


Campos do Formulário
- Full Name 
- Age  
- Course  
- Email  
- Password (mín. 6 caracteres, 1 maiúscula e 1 número)


 Melhorias Futuras
- Criar página separada para listagem  
- Conectar com um backend  
- Exportar registros em PDF/CSV
